Usage:

fingerprint.py [-h] -f SAMPLEFILE -d FDIR --home HOME [-m MODULO]

This program matches a fingerprint with already existing fingerprints

optional arguments:
  -h, --help            show this help message and exit

  -f SAMPLEFILE, --sampleFile SAMPLEFILE
                        File path to sample fingerprint file

  -d FDIR, --fDir FDIR  File path to stored fingerprints

  --home HOME           Directory currently in use
  
  -m MODULO, --modulo MODULO
                        Defines chunk size for folders
